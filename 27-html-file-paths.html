Lección 27 - HTML File Paths

Un File Path (ruta de archivo) indica la ubicación de un archivo dentro de la estructura de carpetas de un sitio web.

1. Ruta absoluta

La imagen se carga utilizando una URL completa.

Código utilizado:

<img src="https://picsum.photos/id/1018/300/200">

2. Ruta relativa - Misma carpeta

Ejemplo:

<img src="paisaje.jpg">

La imagen "paisaje.jpg" debe estar ubicada en la misma carpeta que este archivo HTML.

3. Ruta relativa - Carpeta images

<img src="images/paisaje.jpg">

El navegador buscará la imagen dentro de la carpeta images.

4. Ruta desde la raíz del sitio

<img src="/images/paisaje.jpg">

La búsqueda inicia desde la carpeta principal del sitio web.

5. Ruta hacia la carpeta superior

<img src="../images/paisaje.jpg">

El navegador sube un nivel de carpeta antes de buscar la imagen.

6. Buenas prácticas

  • Utilizar rutas relativas cuando sea posible.
  • Facilitan mover el proyecto entre diferentes computadoras o servidores.
  • Evitan depender de un dominio específico.
  • Funcionan tanto en localhost como en un servidor web.
