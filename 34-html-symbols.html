Lección 34 - HTML Symbols

Los HTML Symbols permiten mostrar símbolos especiales que normalmente no existen en el teclado. Estos símbolos pueden escribirse utilizando:

  • Nombre de la entidad (&copy;)
  • Número decimal (&#169;)
  • Número hexadecimal (&#x00A9;)

1. Símbolos básicos

Los siguientes ejemplos muestran símbolos muy utilizados en HTML.

Símbolo Nombre Entidad Número
© Copyright &copy; &#169;
® Marca registrada &reg; &#174;
™ Trademark &trade; &#8482;
€ Euro &euro; &#8364;

2. Flechas

HTML permite mostrar flechas mediante entidades.

Símbolo Entidad Número
← &larr; &#8592;
↑ &uarr; &#8593;
→ &rarr; &#8594;
↓ &darr; &#8595;

3. Símbolos de cartas

Los siguientes símbolos representan los palos de una baraja.

Símbolo Entidad Número
♠ &spades; &#9824;
♣ &clubs; &#9827;
♥ &hearts; &#9829;
♦ &diams; &#9830;

4. Símbolos matemáticos

HTML también permite mostrar símbolos utilizados en matemáticas.

Símbolo Descripción Entidad Número
∀ Para todo &forall; &#8704;
∂ Derivada parcial &part; &#8706;
∃ Existe &exist; &#8707;
∅ Conjunto vacío &empty; &#8709;
∇ Nabla &nabla; &#8711;
∈ Pertenece a &isin; &#8712;
∉ No pertenece a &notin; &#8713;
∋ Contiene como miembro &ni; &#8715;
∏ Productoria &prod; &#8719;
∑ Sumatoria &sum; &#8721;

5. Letras griegas

Las entidades HTML también permiten mostrar letras del alfabeto griego.

Letra Nombre Entidad Número
Α Alpha &Alpha; &#913;
Β Beta &Beta; &#914;
Γ Gamma &Gamma; &#915;
Δ Delta &Delta; &#916;
Ε Epsilon &Epsilon; &#917;
Ζ Zeta &Zeta; &#918;

6. Ejemplos prácticos

Copyright: © 2026 Mi Sitio Web

Precio: 150 €

Marca: HTML ™

Mover a la derecha: →

Juego de cartas: ♠ ♣ ♥ ♦

Operación matemática: ∑ x = 100

Letras griegas: Α Β Γ Δ

7. Resumen

  • Las entidades HTML permiten mostrar símbolos especiales.
  • Se pueden escribir utilizando un nombre o un número.
  • Existen símbolos para monedas, matemáticas, flechas y letras griegas.
  • Las entidades son compatibles con todos los navegadores modernos.
