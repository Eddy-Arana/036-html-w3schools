Lección 36 - HTML Encoding (Character Sets)

HTML utiliza diferentes conjuntos de caracteres (Character Sets) para representar letras, números, símbolos y caracteres especiales correctamente en cualquier navegador.

1. HTML Charset

Para que una página web muestre correctamente los caracteres, es necesario indicar el conjunto de caracteres mediante la etiqueta meta charset.

<meta charset="UTF-8">

Actualmente, HTML5 recomienda utilizar UTF-8.

2. Conjunto de caracteres ASCII

ASCII fue el primer estándar utilizado en Internet.

  • Letras inglesas (A-Z y a-z).
  • Números (0-9).
  • Símbolos básicos.
  • 128 caracteres en total.
A B C D
a b c d
0 1 2 3
! @ # $ % &

3. Conjunto de caracteres ANSI (Windows-1252)

ANSI fue el primer conjunto de caracteres utilizado por Microsoft Windows.

<meta charset="Windows-1252">
  • Compatible con ASCII.
  • Agrega caracteres especiales.
  • Utilizado en versiones antiguas de Windows.

4. ISO-8859-1

Fue el conjunto de caracteres predeterminado en HTML 4.

<meta charset="ISO-8859-1">
  • Soporta 256 caracteres.
  • Compatible con idiomas europeos.
  • Actualmente ha sido reemplazado por UTF-8.

5. UTF-8

UTF-8 es el estándar recomendado para HTML5.

<meta charset="UTF-8">
  • Compatible con ASCII.
  • Compatible con miles de idiomas.
  • Permite utilizar emojis.
  • Incluye símbolos matemáticos.
  • Incluye caracteres especiales.

6. Comparación de Character Sets

Character Set Cantidad aproximada Uso
ASCII 128 Primer estándar para Internet.
ANSI 256 Windows antiguos.
ISO-8859-1 256 HTML 4.
UTF-8 Más de 1 millón Estándar actual de HTML5.

8. Resumen de la lección

  • HTML necesita un conjunto de caracteres para mostrar correctamente el contenido.
  • El estándar recomendado actualmente es UTF-8.
  • ASCII fue el primer conjunto de caracteres utilizado en Internet.
  • ANSI fue ampliamente utilizado por Microsoft Windows.
  • ISO-8859-1 fue el estándar de HTML 4.
  • UTF-8 soporta millones de caracteres, símbolos y emojis.
  • La etiqueta recomendada es:

<meta charset="UTF-8">
