Lección 33 - HTML Entities

Las entidades HTML permiten mostrar caracteres reservados, símbolos especiales y caracteres que normalmente el navegador interpretaría como código HTML.

1. Caracteres reservados

Para mostrar el símbolo menor que se utiliza: &lt; Resultado: <

Para mostrar el símbolo mayor que: &gt; Resultado: >

Para mostrar un ampersand: &amp; Resultado: &

2. Espacio no separable (&nbsp;)

Sin utilizar &nbsp;:

HTML elimina múltiples espacios.

Utilizando &nbsp;:

HTML    conserva   estos espacios.

4. Entidades usando números

< = <

> = >

© = ©

€ = €

™ = ™

5. Marcas diacríticas

à → a con acento grave

á → a con acento agudo

â → a con acento circunflejo

ã → a con tilde

Ò → O con acento grave

Ó → O con acento agudo

Ô → O con acento circunflejo

Õ → O con tilde

6. Resumen

  • Las entidades HTML permiten mostrar caracteres reservados.
  • Existen entidades por nombre y por número.
  • &nbsp; crea un espacio que no se divide.
  • También permiten mostrar símbolos como ©, ®, ™, €, £ y ¥.
  • Las marcas diacríticas permiten agregar acentos y otros signos a las letras.
