Lección 30 - HTML Responsive Web Design

Ejemplos de diseño adaptable para diferentes tamaños de pantalla.

1. Meta Viewport

Esta página utiliza la etiqueta:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Esto permite que la página se adapte correctamente a cualquier dispositivo.

2. Imagen Responsive usando width:100%

CSS utilizado:

width:100%;

3. Imagen Responsive usando max-width:100%

CSS utilizado:

max-width:100%; height:auto;

4. Elemento Picture

Cambie el tamaño de la ventana para ver cómo cambia la imagen.

5. Texto Responsive usando vw

Hola Mundo

El tamaño del texto cambia automáticamente al modificar el tamaño de la ventana.
