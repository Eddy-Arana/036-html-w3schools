Lección 31 - HTML Computer Code Elements

HTML dispone de varias etiquetas especiales para representar código fuente, entrada del teclado, salida de programas, variables y texto preformateado.

1. Elemento <kbd>

Para guardar un documento presione Ctrl + S

2. Elemento <samp>

Salida de un programa:

Archivo no encontrado.
Presione F1 para continuar.

3. Elemento <code>

Código escrito en una sola línea:

x = 5; y = 6; z = x + y;

4. Elementos <pre> y <code>


int x = 5;
int y = 6;
int z = x + y;

System.out.println(z);

5. Elemento <var>

El área de un triángulo es: 1/2 × b × h

Donde: b = base y h = altura.

6. Resumen

  • <kbd> → Entrada desde el teclado.
  • <samp> → Salida de un programa.
  • <code> → Código fuente.
  • <pre> → Conserva espacios y saltos de línea.
  • <var> → Variables de programación o matemáticas.
