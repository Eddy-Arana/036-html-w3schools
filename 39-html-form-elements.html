Lección 39 - HTML Form Elements

Esta lección muestra todos los elementos que pueden formar parte de un formulario HTML.

1. Elemento <input>

Es el elemento más utilizado dentro de un formulario.


2. Elemento <label>

El elemento <label> describe un campo del formulario y mejora la accesibilidad.


3. Elemento <select>

Permite seleccionar una opción de una lista.


4. Opción seleccionada por defecto

5. Atributo size

Muestra varias opciones visibles al mismo tiempo.

6. Selección múltiple

Con el atributo multiple se pueden seleccionar varias opciones.

7. Elemento <textarea>

Permite escribir texto en varias líneas.

8. Textarea usando CSS

9. Elemento <button>

Botón que ejecuta JavaScript.

10. Fieldset y Legend

Agrupan elementos relacionados dentro del formulario.

Información Personal



7. <fieldset> y <legend>

<fieldset> agrupa controles relacionados y <legend> coloca un título.

Información Personal





8. <datalist>

Permite mostrar sugerencias mientras el usuario escribe.


9. <output>

Muestra el resultado de un cálculo realizado con JavaScript.

10. Resumen

Elemento Función
<form> Contenedor del formulario.
<input> Campo de entrada.
<label> Etiqueta descriptiva.
<select> Lista desplegable.
<option> Opción dentro del select.
<textarea> Área de texto.
<button> Botón.
<fieldset> Agrupa controles.
<legend> Título del grupo.
<datalist> Sugerencias para un input.
<output> Muestra resultados.
