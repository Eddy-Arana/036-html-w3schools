Lección 40 - HTML Input Types

El elemento <input> puede comportarse de muchas formas dependiendo del atributo type.

1. Input type="text"

Campo de texto de una sola línea.


2. Input type="password"

Oculta los caracteres escritos.


3. Input type="submit"

Envía el formulario.



4. Input type="reset"

Restablece todos los valores del formulario.



5. Input type="radio"

Permite seleccionar solo una opción.

7. Input type="button"

Botón que ejecuta una acción.

8. Input type="color"

Selector de color.

9. Input type="date"

Permite seleccionar una fecha.

10. Input type="datetime-local"

Selecciona fecha y hora.

11. Input type="email"

Campo para direcciones de correo.

12. Input type="file"

Permite seleccionar archivos.

13. Input type="hidden"

Este campo permanece oculto para el usuario.

14. Input type="month"

Permite seleccionar un mes y año.


15. Input type="number"

Campo para introducir números.


Restricciones de Input

Los campos pueden tener restricciones usando atributos.

  • min: valor mínimo permitido
  • max: valor máximo permitido
  • step: intervalo de números
  • required: campo obligatorio
  • readonly: solo lectura
  • disabled: campo deshabilitado

16. Input type="range"

Crea un control deslizante.


17. Input type="search"

Campo utilizado para búsquedas.


18. Input type="tel"

Campo para números telefónicos.


19. Input type="time"

Permite seleccionar una hora.


20. Input type="url"

Campo para direcciones web.


21. Input type="week"

Permite seleccionar una semana y año.
