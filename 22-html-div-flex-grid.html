Resumen de la Lección 22

  • El elemento <div> es un contenedor de tipo Block.
  • Se utiliza para organizar secciones de una página web.
  • Puede contener cualquier elemento HTML.
  • Puede alinearse utilizando float.
  • También puede alinearse con display:inline-block.
  • Actualmente la forma recomendada es utilizar Flexbox.
  • Para diseños más complejos se recomienda CSS Grid.
