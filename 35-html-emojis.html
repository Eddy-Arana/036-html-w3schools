Lección 35 - HTML Emojis

Los Emojis son caracteres pertenecientes al conjunto de caracteres UTF-8 (Unicode). No son imágenes, sino caracteres especiales que pueden mostrarse como cualquier otro texto dentro de una página HTML.

1. Configuración UTF-8

Para que los emojis y otros caracteres especiales funcionen correctamente, debemos incluir la siguiente etiqueta dentro del elemento <head>.

<meta charset="UTF-8">

2. Caracteres UTF-8

Los caracteres también pueden representarse mediante números.

Carácter Número
A &#65;
B &#66;
C &#67;
Resultado A B C

3. Primer Emoji

23. Resumen

  • Los emojis pertenecen al conjunto de caracteres UTF-8 (Unicode).
  • La etiqueta <meta charset="UTF-8"> permite mostrarlos correctamente.
  • Los emojis pueden escribirse directamente o mediante su número Unicode.
  • Al ser caracteres de texto, pueden cambiar de tamaño usando CSS.
  • HTML permite utilizar miles de emojis para mejorar la apariencia de una página web.
