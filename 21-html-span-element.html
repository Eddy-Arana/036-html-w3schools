El elemento <span>

El elemento span es un contenedor de tipo Inline. Se utiliza para aplicar estilos a una parte específica de un texto sin afectar el resto del contenido.

Ejemplo básico

Mi madre tiene ojos azules y mi padre tiene ojos verdes .

Resaltando palabras

HTML significa HyperText Markup Language , el lenguaje utilizado para crear páginas web.

Resumen de la lección

  • Los elementos Block comienzan en una nueva línea.
  • Los elementos Block ocupan todo el ancho disponible.
  • Los elementos Inline permanecen en la misma línea.
  • Los elementos Inline solo ocupan el espacio necesario.
  • El elemento <div> es un contenedor de tipo Block.
  • El elemento <span> es un contenedor de tipo Inline.
  • Ambos elementos se utilizan frecuentemente junto con CSS.
