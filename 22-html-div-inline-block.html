Div utilizando display: inline-block

La propiedad display:inline-block permite mostrar varios elementos uno junto al otro, conservando el comportamiento de un elemento en línea.

¿Qué hace inline-block?

  • Permite colocar varios elementos en una misma fila.
  • No es necesario utilizar float.
  • Los elementos conservan su ancho y alto.
  • Es útil para crear diseños sencillos.
  • Actualmente suele reemplazarse por Flexbox o CSS Grid en proyectos modernos.
