Lección 28 - HTML Head Element

El elemento <head> contiene información sobre el documento HTML que no se muestra directamente en la página.

1. Elemento <title>

El título de esta página aparece en la pestaña del navegador.

2. Elemento <style>

Los estilos CSS utilizados en esta página fueron definidos dentro del elemento <style>.

3. Elementos <meta>

  • Charset UTF-8.
  • Viewport para dispositivos móviles.
  • Descripción de la página.
  • Palabras clave.
  • Autor del documento.

4. Elemento <link>

Se utiliza para conectar archivos externos, normalmente hojas de estilo CSS.

<link rel="stylesheet" href="estilos.css">

5. Elemento <script>

Presione el botón.

6. Elemento <base>

El elemento <base> establece una URL base para todos los enlaces relativos de la página.

<base href="https://www.w3schools.com/" target="_blank">
