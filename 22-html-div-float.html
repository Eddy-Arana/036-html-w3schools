¿Qué hace cada propiedad?

  • float:left; coloca los elementos horizontalmente.
  • width:33%; hace que cada div ocupe aproximadamente un tercio del espacio.
  • overflow:auto; permite que el contenedor principal incluya correctamente los elementos flotantes.
  • box-sizing:border-box; hace que el padding no aumente el ancho total del div.
