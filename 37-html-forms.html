Lección 37 - HTML Forms

Los formularios HTML permiten recopilar información del usuario como nombres, correos electrónicos, opciones seleccionadas y otros datos para enviarlos a un servidor.

1. Elemento <form>

El elemento <form> es el contenedor de todos los controles del formulario.

<form> ... </form>

Este formulario contiene diferentes elementos HTML.

2. Campos de texto (Text Fields)

Los campos de texto permiten ingresar información en una sola línea.




3. Elemento <label>

La etiqueta <label> identifica claramente cada campo del formulario.

El atributo for debe coincidir con el atributo id del elemento correspondiente.

<label for="correo">Correo</label>


4. Radio Buttons

Los botones de opción permiten seleccionar solamente una opción.



5. Checkboxes

Las casillas de verificación permiten seleccionar cero, una o varias opciones.



6. Botón Submit

El botón Submit envía la información del formulario al servidor indicado en el atributo action.







7. Atributo name

Cada campo del formulario debe tener un atributo name, ya que este identifica el dato que será enviado al servidor.

<input type="text" name="nombre">

Si un campo no posee el atributo name, su información no será enviada cuando el usuario presione el botón Submit.

9. Resumen de la lección

  • El elemento <form> crea un formulario HTML.
  • <input> permite ingresar información.
  • <label> identifica cada campo del formulario.
  • Los Radio Buttons permiten seleccionar una sola opción.
  • Los Checkboxes permiten seleccionar varias opciones.
  • El botón Submit envía los datos al servidor.
  • El atributo name es obligatorio para enviar los datos.
