Elementos Block e Inline

Elementos de Bloque

Un elemento de bloque siempre comienza en una nueva línea y ocupa todo el ancho disponible.

Hola Mundo desde un elemento <p>

Hola Mundo desde un elemento <div>

Ambos elementos ocupan todo el ancho disponible del navegador.

Elementos Inline

Este texto contiene un elemento span dentro del mismo párrafo.

El elemento <span> NO inicia una nueva línea y solamente ocupa el espacio necesario.

Diferencias

Un elemento Block comienza en una nueva línea.

Un elemento Block ocupa todo el ancho disponible.

Un elemento Inline permanece en la misma línea.

Un elemento Inline solo ocupa el espacio que necesita.
